# La Gambita Alicantina

Sitio web del restaurante La Gambita Alicantina, especializado en marisco fresco, arroces melosos y cocina mediterránea en Alicante.

## Características

- Diseño responsive para todos los dispositivos
- Información del menú y especialidades
- Sistema de reservas online
- Galería de platos
- Información de contacto y horarios

## Tecnologías

- React 18 con TypeScript
- Vite como bundler
- Tailwind CSS para estilos
- Radix UI para componentes accesibles

## Desarrollo

```bash
npm install
npm run dev
```

El servidor de desarrollo se iniciará en http://localhost:8080
